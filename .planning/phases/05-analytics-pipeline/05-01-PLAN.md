## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Cloud Run Job SA → Supabase service role + BigQuery write | Batch path can read all ops tables; must project allowlisted columns only. |
| Officer client → FastAPI GET /api/v1/analytics | Untrusted query params (from/to); JWT required. |
| FastAPI → BigQuery | Server-side only; credentials must never reach the browser. |
| Scheduler → Job :run | Abusable if publicly invokable without IAM. |
