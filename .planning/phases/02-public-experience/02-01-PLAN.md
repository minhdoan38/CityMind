## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Citizen browser → Next BFF → FastAPI | Untrusted multipart report + optional image; client-controlled XFF until platform hop. |
| FastAPI → Supabase service role | Privileged write of report + token_hash; must never expose secret key or plaintext token after response. |
